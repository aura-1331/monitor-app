PowerSupply Monitor
A lightweight, real-time system utility designed to track power consumption, battery health, and voltage stability for Windows-based environments.

Core Features
Real-Time Analytics: Monitor wattage draw, voltage fluctuations, and current amperage.

Battery Diagnostics: View design capacity versus current capacity and cycle counts.

Automated Alerts: Receive notifications for critical power drops or overheating.

Logging: Export power usage history to CSV for long-term performance analysis.

Background Execution: Runs as a silent .pyw process with minimal system overhead.

Getting Started
Prerequisites
Python: 3.12+

OS: Windows 10/11

Permissions: Administrator access (required for hardware sensor data)
